POLE BOT - Idea del Proyecto

Pole Bot es un bot de Discord que consiste en escribir el primer mensaje en un canal o hilo.

Funcionalidad:
- El bot detecta cuando se crea un nuevo canal o hilo
- Automáticamente escribe el primer mensaje (hace "pole")
- Es un bot simple y divertido basado en la tradición de Internet de ser el primero en comentar

CAMBIOS PARA LA FUNCIONALIDAD

Podemos hacer que el pole sea DIARIO, reset a las 12h, y si no es escribe pole a las 12, penalty (por pensar eso)

## 🔔 Configuración de Ping de Apertura (Nuevo Requisito v1.0)

Objetivo: permitir configurar qué rol se menciona cuando se abre el pole diario.

Estados posibles:
- `none` (no se menciona nada)
- `role:<id>` (rol específico configurado)
- `everyone` (usa @everyone, requiere confirmación)

Flujo @everyone:
1. Admin ejecuta `/poleping set everyone`
2. Bot responde con advertencia: "¿Seguro? Puede ser spam." y requiere `/poleping confirm` dentro de 60s.
3. Si confirma, se guarda.
4. Si no, se cancela.

Comandos plan:
- `/poleping set <@rol|everyone|none>`
- `/poleping show` (muestra configuración actual)
- Integrar alternativa en `/config` con botón "Editar Ping".

Persistencia:
- Campo nuevo en configuración de servidor: `ping_mode` (none|role|everyone)
- Campo `ping_role_id` (nullable)
