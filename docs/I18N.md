# 🌐 i18n (Internacionalización)

Pole Bot soporta idioma por servidor.

## 🗣️ Idiomas soportados

- `es` (default)
- `en`

## 📍 Ubicación

- Traducciones: `utils/i18n.py`
- Idioma por servidor: `servers.language`
- Schema actual: `v8`

## 🧩 Uso en código

```python
from utils.i18n import t

msg = t('settings.title', guild_id)
msg2 = t('pole.critical', guild_id, minutes=5)
```

## ✅ Reglas obligatorias

- No hardcodear mensajes de usuario en cogs.
- Toda key nueva debe existir en `es` y `en`.
- Mantener placeholders consistentes entre idiomas.

## 🎲 Variaciones aleatorias

Si una key es lista, `t()` selecciona una opción aleatoria automáticamente.

## 🧪 Scripts de validación

```bash
python scripts/check_missing_translations.py
python scripts/check_placeholders.py
```
